# CalculadoraIMC

Este es un programa en Java que calcula el Índice de Masa Corporal (IMC) de una persona. 
El usuario ingresa su peso en kilogramos y su altura en metros, y el programa calcula el IMC y lo muestra redondeado a dos decimales.
## Cómo ejecutarlo

1. Compilar el código:

```bash
javac CalculadoraIMC.java

### Ejecutarlo
java CalculadoraIMC
